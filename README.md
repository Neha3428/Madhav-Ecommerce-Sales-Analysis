## Madhav Store Sales Dashboard | Power BI Project
# Overview
This project is an interactive Sales Dashboard developed in Power BI to analyze and visualize sales performance across different states, customers, product categories, and payment modes.

The dashboard provides actionable business insights by transforming raw sales data into meaningful visual reports, helping stakeholders monitor performance and make data-driven decisions.
## Project Objective
Analyze overall sales and profit performance,
Identify top-performing states and customers,
Evaluate category-wise and sub-category-wise sales,
Understand customer purchasing behavior,
Track monthly profit trends,
Build an interactive dashboard for business decision-making.
## Tools & Technologies Used
Microsoft Power BI,
Power Query,
DAX (Data Analysis Expressions),
Data Modeling,
Data Visualization.

## Dataset Information
Order Details,
Customer Information,
Product Categories,
Sub-Categories,
State-wise Sales Data,
Payment Modes,
Quantity Sold,
Profit and Revenue Metrics.

## Key Insights
Identified high-performing states contributing maximum revenue,
Analyzed customer segments driving business growth,
Found the most profitable product sub-categories,
Evaluated monthly profit fluctuations and sales trends,
Compared performance across different payment methods.
